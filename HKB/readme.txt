# 🔷 PREDIKTOR 6 ANGKA - PANDUAN INSTALASI

## 📋 DAFTAR FILE INSTALASI

### Untuk Python Biasa
| File | Platform | Kegunaan |
|------|----------|----------|
| `install_python.bat` | Windows | Instalasi otomatis untuk Python di Windows |
| `install_python.sh` | Linux/Mac | Instalasi otomatis untuk Python di Linux/Mac |

### Untuk Anaconda
| File | Platform | Kegunaan |
|------|----------|----------|
| `install_anaconda.bat` | Windows | Instalasi otomatis untuk Anaconda di Windows |
| `install_anaconda.sh` | Linux/Mac | Instalasi otomatis untuk Anaconda di Linux/Mac |

---

## 🚀 CARA PENGGUNAAN

### **Windows**
1. Double-click file `.bat` yang sesuai
2. Ikuti petunjuk di layar
3. Setelah selesai, jalankan `run_prediktor.bat`

### **Linux/Mac**
```bash
# Beri izin execute
chmod +x install_python.sh
# atau
chmod +x install_anaconda.sh

# Jalankan
./install_python.sh
# atau
./install_anaconda.sh

# Setelah selesai
./run_prediktor.sh
